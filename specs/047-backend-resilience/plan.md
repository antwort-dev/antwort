# Implementation Plan: Backend Resilience (Circuit Breaker + Retry)

**Branch**: `047-backend-resilience` | **Date**: 2026-03-29 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/047-backend-resilience/spec.md`

## Summary

Add a resilience layer that wraps any `provider.Provider` with circuit breaker protection and configurable retry with exponential backoff. The wrapper is transparent to the engine, opt-in via a global `resilience:` config block, and integrates with existing Prometheus metrics and debug logging infrastructure. Special handling for 429 responses respects `Retry-After` headers without tripping the circuit breaker.

## Design Details

### ResilientProvider (Decorator)

```
ResilientProvider implements provider.Provider
  ├── inner provider.Provider    (the wrapped provider)
  ├── cb    *CircuitBreaker      (per-instance state machine)
  ├── retry RetryPolicy          (immutable config)
  └── Methods:
      ├── Name()         → delegates to inner
      ├── Capabilities() → delegates to inner
      ├── ListModels()   → delegates to inner (no resilience)
      ├── Close()        → delegates to inner
      ├── Complete()     → retry loop with circuit breaker
      └── Stream()       → retry loop with circuit breaker (connection phase only)
```

### Complete() Flow

```
Complete(ctx, req):
  for attempt = 1..maxAttempts:
    if !cb.Allow():
      return CircuitOpenError
    resp, err = inner.Complete(ctx, req)
    if err == nil:
      cb.RecordSuccess()
      return resp, nil
    classification = classify(err)
    if classification == RateLimited:
      wait = parseRetryAfter(err) or backoff(attempt)
      if wait > contextDeadline:
        return err
      sleep(ctx, wait)
      continue  // no circuit breaker impact
    if classification == NonRetryable:
      return err  // no circuit breaker impact, no retry
    cb.RecordFailure()
    if attempt < maxAttempts:
      sleep(ctx, backoff(attempt))
    else:
      metrics.RetryExhausted++
      return err
```

### Stream() Flow

```
Stream(ctx, req):
  for attempt = 1..maxAttempts:
    if !cb.Allow():
      return nil, CircuitOpenError
    ch, err = inner.Stream(ctx, req)
    if err == nil:
      cb.RecordSuccess()
      return ch, nil  // channel returned, no more retries
    // Same error handling as Complete()
    ...
```

### Configuration Wiring

In `cmd/server/main.go`:
```
prov, err := createProvider(cfg)
```
